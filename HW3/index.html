Overview

This assignment implements a two-view calibrated SfM pipeline on the staff-provided image pair: camera intrinsics estimation, SIFT feature detection, NNDR feature matching, RANSAC pose estimation via the Essential matrix (8-point), and sparse triangulation with reprojection filtering.
